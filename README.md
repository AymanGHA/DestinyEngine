# DestinyEngine
Minimal 2D/3D game engine which is focused to make rts games . 

- Features :
  * Unity Engine sintax
  * Entity - component - System (ECS)
  * Networking
  * Tools -> Shader editor , Pixel Editor , AutoTile , Tilemap , Image Converters
 
- Basic Geometry - Cube - Plane - Sphere
- Shader Collection ;
